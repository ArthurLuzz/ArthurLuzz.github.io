Profissão Desenvolvedor Web

Desenvolvedor Front-end e Back-end, Arquitetura, Ferramentas e Linguagens.

Frontend Developer

O Front-End é onde encontramos a interface de navegação de um site, com o design da página e formulários de contato, por exemplo. A estrutura do Front-End é composta geralmente por HTML, CSS, Javascript e Flash.

Backend Developer

O Back-End é o nível de programação que envolve banco de dados. MySQL, MS-SQL, Oracle e SAP e fazem parte do Back-End, que é responsável pelo processamento final dos dados recebidos do Front-End.

DevOps Roadmap

DevOps é uma prática de engenharia de software que possui o intuito de unificar o desenvolvimento de software (Dev) e a operação de software.
